# PDFスライドビューア

動画リンク・目次・タイマー機能付きのブラウザベースPDFスライドビューアです💖

## ✨ 主要機能

- **PDFスライド表示** - フルスクリーン対応、高解像度レンダリング
- **動画埋め込み** - ローカル動画・YouTube対応、モーダル再生
- **ナビゲーション** - 目次・進捗バー・スライド間リンク
- **便利機能** - タイマー・印刷・URL直接アクセス
- **操作性** - キーボード（矢印・PageUp/Down）・マウス・タッチ対応

## 🎯 コンセプト

**ビューアアプリとスライドデータを完全分離**

- `viewer/` - 開発者が配布・更新するビューアアプリ
- `slides/` - スライド作成者が管理するプレゼンデータ
- ビューア更新時も、スライドデータはそのまま！

## 🚀 クイックスタート

### 開発者向け

```bash
npm install      # 依存パッケージのインストール
npm run dev      # 開発サーバー起動（http://localhost:3000）
npm run build    # 配布用ビルド（dist/フォルダに出力）
```

詳しくは → [開発者ガイド](docs/development.md)

### スライド作成者向け

1. `dist/viewer/` フォルダをコピー
2. 同じ階層に `slides/001/` フォルダを作成
3. `config.json` と PDFファイルを配置
4. `viewer/index.html` を開く

詳しくは → [スライド作成者ガイド](docs/creators-guide.md)
